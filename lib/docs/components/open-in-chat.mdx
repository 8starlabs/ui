## Examples

### Default Providers

By default, `OpenInChat` renders menu items for ChatGPT, Claude, Perplexity,
Gemini, and Grok. Each item opens the selected provider in a new tab with the
component prompt encoded into the provider's prompt query parameter.

<ComponentPreview name="open-in-chat-demo" />

### Custom Providers

Pass provider-specific link components as children when you only want to expose
some chat tools or control their order. Child links inherit the parent prompt,
and can override it with their own `prompt` prop.

<ComponentPreview name="open-in-chat-custom-providers-demo" />

### Custom Link

Use `OpenInChatLink` when you need to add a provider that is not included in the
default list. It accepts a label, base URL, prompt parameter name, icon, and
optional per-link prompt override.

<ComponentPreview name="open-in-chat-custom-link-demo" />

### Positioning And Styling

Use `align` and `side` to position the dropdown content, and use
`triggerClassName` or `contentClassName` to adjust the trigger and menu.

```tsx showLineNumbers
<OpenInChat
  prompt="Turn this changelog into release notes."
  triggerLabel="Ask AI"
  align="start"
  side="bottom"
  triggerClassName="w-full justify-between"
  contentClassName="w-64"
/>
```

## API Reference

### OpenInChat

| Prop               | Type                                        | Default          | Description                                                                 |
| ------------------ | ------------------------------------------- | ---------------- | --------------------------------------------------------------------------- |
| `prompt`           | `string`                                    | -                | **Required.** The prompt sent to each generated chat link.                  |
| `children`         | `React.ReactNode`                           | -                | Optional. Custom provider links to render instead of the default list.      |
| `triggerLabel`     | `React.ReactNode`                           | `"Open in chat"` | Optional. Content rendered before the chevron in the trigger button.        |
| `triggerClassName` | `string`                                    | -                | Optional. Additional classes applied to the button trigger.                 |
| `contentClassName` | `string`                                    | -                | Optional. Additional classes applied to the dropdown content.               |
| `align`            | `DropdownMenuContent["align"]`              | `"end"`          | Optional. Aligns the dropdown content relative to the trigger.              |
| `side`             | `DropdownMenuContent["side"]`               | -                | Optional. Controls which side of the trigger the dropdown opens from.       |
| `...props`         | `React.ComponentProps<typeof DropdownMenu>` | -                | Optional. Additional props passed to the underlying `DropdownMenu` element. |

### Provider Links

The package exports provider-specific link components for composing a custom
menu:

- `OpenInChatGPT`
- `OpenInClaude`
- `OpenInPerplexity`
- `OpenInGemini`
- `OpenInGrok`

| Prop        | Type     | Default                    | Description                                                   |
| ----------- | -------- | -------------------------- | ------------------------------------------------------------- |
| `prompt`    | `string` | Parent `OpenInChat` prompt | Optional. Overrides the parent prompt for this provider link. |
| `className` | `string` | -                          | Optional. Additional classes applied to the dropdown item.    |

### OpenInChatLink

| Prop        | Type                | Default                    | Description                                                 |
| ----------- | ------------------- | -------------------------- | ----------------------------------------------------------- |
| `baseUrl`   | `string`            | -                          | **Required.** Base URL for the custom chat destination.     |
| `paramName` | `string`            | `"q"`                      | Optional. Query parameter used for the prompt.              |
| `icon`      | `React.ElementType` | -                          | **Required.** Icon component rendered before the label.     |
| `label`     | `string`            | -                          | **Required.** Label rendered as `Open in {label}`.          |
| `prompt`    | `string`            | Parent `OpenInChat` prompt | Optional. Overrides the parent prompt for this custom link. |
| `className` | `string`            | -                          | Optional. Additional classes applied to the dropdown item.  |

### aiLinksDataMap

`aiLinksDataMap` is exported for consumers who need the built-in provider
metadata. It contains each provider's base URL, optional prompt parameter, and icon
component.
