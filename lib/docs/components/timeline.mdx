## Usage

```tsx showLineNumbers
import {
  Timeline,
  TimelineItem,
  TimelineItemDate,
  TimelineItemTitle,
  TimelineItemDescription
} from "@/components/ui/timeline";

export function MyTimeline() {
  return (
    <Timeline orientation="horizontal">
      <TimelineItem>
        <TimelineItemDate>January 2024</TimelineItemDate>
        <TimelineItemTitle>Project Kickoff</TimelineItemTitle>
        <TimelineItemDescription>
          Initial planning and requirements gathering.
        </TimelineItemDescription>
      </TimelineItem>

      <TimelineItem variant="secondary">
        <TimelineItemDate>February 2024</TimelineItemDate>
        <TimelineItemTitle>Design Phase</TimelineItemTitle>
        <TimelineItemDescription>
          Mockups and prototypes approved.
        </TimelineItemDescription>
      </TimelineItem>
    </Timeline>
  );
}
```

## API Reference

`<Timeline>` props

| Prop               | Type                           | Default        | Description                                                                                                                                                                                                             |
| ------------------ | ------------------------------ | -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `orientation`      | `"horizontal" \| "vertical"`   | `"horizontal"` | The orientation of the timeline.                                                                                                                                                                                        |
| `alternating`      | `boolean`                      | `true`         | Whether the timeline cards alternate from one side of the line to the other. If set to `true`, `alignment` has no effect. The first card is always placed at the top left (for both horizontal and vertical timelines). |
| `alignment`        | `"top/left" \| "bottom/right"` | `"top/left"`   | Determines which side of the line to display the cards. Only takes effect if `alternating` is `false`.                                                                                                                  |
| `horizItemSpacing` | `number`                       | `130`          | The horizontal distance (in px) between successive timeline points on a horizontal timeline.                                                                                                                            |
| `horizItemWidth`   | `number`                       | `220`          | The width (in px) of each card in a horizontal timeline.                                                                                                                                                                |
| `vertItemSpacing`  | `number`                       | `130`          | The vertical distance (in px) between successive timeline points on a vertical timeline.                                                                                                                                |
| `vertItemMaxWidth` | `number`                       | `350`          | The maximum width (in px) of each card in a vertical timeline.                                                                                                                                                          |
| `noCards`          | `boolean`                      | `false`        | If `true`, removes the card background and border styling from all items.                                                                                                                                               |

`<TimelineItem>` props

| Prop      | Type                                                   | Description                                               |
| --------- | ------------------------------------------------------ | --------------------------------------------------------- |
| `variant` | `"default" \| "success" \| "destructive" \| "outline"` | (Optional) The visual style of the timeline card and dot. |
