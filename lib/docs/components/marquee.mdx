## API Reference

| Prop           | Type                | Default  | Description                                                         |
| -------------- | ------------------- | -------- | ------------------------------------------------------------------- |
| `grayscale`    | `boolean`           | `false`  | Optional. Applies grayscale filter to the marquee content.          |
| `direction`    | `"left" \| "right"` | `"left"` | Optional. Controls the direction of the scrolling animation.        |
| `pauseOnHover` | `boolean`           | `true`   | Optional. If set to true, pauses the marquee on hover.              |
| `fade`         | `boolean`           | `true`   | Optional. Shows gradient fade overlays on the left and right edges. |
| `children`     | `React.ReactNode`   | -        | Optional. The content to be animated in the marquee.                |
